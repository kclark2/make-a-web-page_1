HTML Basics

What is HTML

- HTMl is the language of the web, and has syntax and rules.
- HTML code acts as a website's structure or framework (the walls of a metaphorical house).
- While not all elements are visible (ex: 'head' elements are invisible), all visible elements are rectangular.

HTML Tags and Textual Effects

Tags are a basic HTML 'word' that the browser turns into elements in the 'tree.' Common tags include:
- 'p' : Designates a box around a 'paragraph,' or rather a block of text grouped together.
- 'b' : Bolds text.
- 'em': Used to italicize text.
- 'a href=' : Allows you to link other pages' content.
- 'div' : Defines code in a box (boxes and boxes and boxes). 'Divs' are 'block' elements (instead of 'inline') and therefore by default, take up the entire width of the page.

The Importance of Structured Code

If your code is properly structured it is: 1) Easier for programmers to read, and 2) Easier to see if you've made any mistakes.

Having structured code allows programmers to easily see how certain sections of code relate to others (ex: 'x' is the overarching theme, 'y,' and 'y-description' are nested under 'x,' and so we can infer that they are conceptually related to 'x').

Intro to HTML + CSS

CSS: Adding Style to Code

CSS, or 'Cascading Style Sheets,' is another kind of code that gives your HTML (structure) code some style. If HTML is the walls of your house, CSS is the paint and furnishings. CSS can use mulitple sheets to determine style in HTML. It's like law. There's the National, State, Municipality, and 'house' laws. However, CSS is slightly different. Unlike in world law, in CSS the most specific rule is applied to every element (so 'house rules' are basically equiv. to real world National law).

Using CSS code also helps you to avoid repetition in your code which not only saves you time/effort, but also decreases the chance of there being error in your code. While you can apply 'style' directly to your HTML code, doing so is highy repetitive and is really only good for simple/short code projects due to the high chance of error.

Giving Elements Class Attributes

Class attributes (ex: 'div class='whatever'') allow you to give your HTML code 'style' by acting as a reference point for the CSS code. They are like labels on moving boxes that tell you where everything belongs.
